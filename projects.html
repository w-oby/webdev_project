JokeAPI

This project is a Python application that sets up a local API to serve random jokes to a CLI (Command Line Interface) window. The user can setup and launch the application with only two clicks. It features the usage of a Flask server, a CLI window, automated setup and startup procedures, and a Pytest testsuite.

Check out JokeAPI and have a few laughs!

Headless React & Laravel API

This project demonstrates the setup and usage of a headless web application, using both React and a Laravel API. The API consists of Tour de France themed testing data to best mirror the actual usecase of headless applications.

Check out both the React and Laravel repositories.

C# Order System

C# Order System is a project where I have developed the logic of a realistic product ordering, sales and shipping system. The project ensures that the different components interact with each other as smoothly as possible.

Find out how the Order System works!
